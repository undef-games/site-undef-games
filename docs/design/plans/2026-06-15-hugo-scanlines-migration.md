# Hugo Scanlines Migration Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Convert `undef-logos` into a Hugo site using the private `scanlines` theme while preserving the current Vite/React/Pixi logo lab at `/lab/`.

**Architecture:** Hugo owns the public landing page at `/`, using repo-local theme files in `themes/scanlines`. The existing React/Pixi app moves into `lab/`, keeps its own package and tests, and builds with Vite base `/lab/` so the output can be copied into `public/lab/`. The root Makefile creates one Cloudflare Pages artifact in `public/`.

**Tech Stack:** Hugo, Vite, React, PixiJS, TypeScript, Vitest, Playwright, Wrangler, Terraform, Make.

---

- [ ] **Step 12: Create minimal Hugo scanline JavaScript**

Create `themes/scanlines/static/js/scanlines.js`:

```js
const root = document.documentElement

function updateScroll() {
  const maxScroll = Math.max(1, document.documentElement.scrollHeight - window.innerHeight)
  root.style.setProperty('--scan-scroll', String(Math.min(1, Math.max(0, window.scrollY / maxScroll))))
}

function updatePointer(event) {
  root.style.setProperty('--scan-pointer-x', String((event.clientX / Math.max(1, window.innerWidth) - 0.5).toFixed(4)))
}

window.addEventListener('scroll', updateScroll, { passive: true })
window.addEventListener('resize', updateScroll)
window.addEventListener('pointermove', updatePointer, { passive: true })
updateScroll()
```

- [ ] **Step 1: Create build stamp script**

Create `scripts/stamp_build.sh`:

```bash
#!/usr/bin/env bash
set -euo pipefail

ROOT_DIR=$(cd "$(dirname "$0")/.." && pwd -P)
cd "$ROOT_DIR"

SHA=$(git rev-parse --short HEAD 2>/dev/null || echo "local")
DATE_UTC=$(date -u +%Y.%m.%d)
ISO_UTC=$(date -u +%Y-%m-%dT%H:%M:%SZ)

mkdir -p data
cat > data/build.json <<EOF
{
  "version": "v${DATE_UTC}.${SHA}",
  "sha": "${SHA}",
  "date": "${DATE_UTC}",
  "iso": "${ISO_UTC}"
}
EOF

echo "build stamp: v${DATE_UTC}.${SHA} (${ISO_UTC})"
```

Run:

```bash
chmod +x scripts/stamp_build.sh
```
