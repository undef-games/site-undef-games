# Playable Logo Instruments Design

## Goal

Replace the shallow logo gallery with three small interactive identity systems. Each logo state must be earned through a concrete action: define rules, run commands, or route a board token.

## Visual Thesis

Undef Games is a black-box game lab: physical tokens, terminal output, and clean identity specimens. The mark should feel generated by play, not decorated by abstract motion.

## Concepts

### Define the Game

The undefined mark resolves when the user defines three rules: world, move, and win. Each defined rule locks a fragment into the symbol. The final state becomes a stable play/UG mark.

### Command Console

The terminal is the main object. The user runs commands in order: `define world`, `spawn player`, and `run game`. Valid commands compile the identity; invalid commands produce a visible error/glitch state without advancing progress.

### Rule Board

The user advances a token across a small board route. A deliberate illegal move reshapes the board, then legal route steps lock the mark. The final logo is the route produced by play.

## Interaction Requirements

- The main scene click/tap advances the active concept's next valid step.
- The side panel exposes concept-specific controls, not generic phase numbers.
- The specimen panel reflects the same progress as the scene.
- Switching concepts preserves each concept's progress.
- Reset clears only the active concept.

## Implementation Shape

- Add a pure `logo-play-state` module for progress, phase derivation, and reducers.
- Update `AppShell` to own `LogoPlayState` and command input.
- Update `LogoLabScene`, `LogoMark`, `LogoWordmark`, and `ControlPanel` to render from progress.
- Reduce weak prompt/specimen clutter so the canvas and current instrument are the dominant first-screen objects.

## Verification

- Unit tests cover pure state reducers.
- React tests cover all three concept loops through visible controls.
- E2E covers a representative live flow: define rules, run console command, route board, mobile smoke.
